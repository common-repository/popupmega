<strong>
Get mega popup one of the most advanced WordPress Popup plugin now.  Increase your subscribers and sales by more than 500%.
</strong>

There are no major functionality compromises made in the free version of the plugin but you have to purchase the <a href="http://rocketplugins.com/popup-revolution/">premium version</a> of the plugin to get extra rules, extra email services are supported like aweber, campaign monitor and themes like HTML theme which will let you show any kind of valid HTML content inside the popup be it videos, forms, images etc. Two really beautiful and high converting popup themes with subscriber forms are included in the <a href="http://rocketplugins.com/popup-revolution/">premium version</a>


<div>
  <strong>Features</strong>
</div>

*   Can create link popups which trigger popup after clicking on a link or image
*   Mailchimp is supported ( Aweber, Campaign monitor are in premium version )
*   Can set the delay time after which popup will appear or Popup scheduling
*   Can change the mask color
*   Option to hide the popup on mobile devices
*   Option to show the **popup only** to search engine visitors
*   Option to hide the popup on the homepage
*   Option to show the **popup only** on the homepage
*   Option to hide the popup for logged-in users
